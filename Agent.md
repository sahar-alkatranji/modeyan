## ✅ أفضل الممارسات المكتشفة
- **فصل المكونات تفادياً لإعادة التأسيس (Re-mounting):** تعريف أي مكون (مثل `CustomizationView`) داخل مكون أبوي يؤدي لإعادة رندرة المكون بالكامل وفقدان التركيز (Focus) مع كل ضغطة زر أو إدخال نصي. يجب تعريفه دائماً في ملف مستقل.
- **إدارة التثبيت:** استخدام `--legacy-peer-deps` عند عمل `npm install` لتجنب تعارضات React 19 مع مكتبة Stripe القديمة نسبياً.
